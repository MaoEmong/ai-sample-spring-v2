# layout

## 목적
애플리케이션 전반에 공통으로 사용되는 웹 페이지 레이아웃 템플릿입니다.

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `header.mustache` | 모든 페이지 상단에 노출되는 네비게이션 바 및 공통 CSS 참조를 포함하는 헤더 |
| `footer.mustache` | 모든 페이지 하단에 노출되는 공통 정보 및 스크립트를 포함하는 푸터 |

## 하위 디렉토리
- 없음

## AI 작업 지침
- **템플릿 상속**: Mustache의 부분(partial) 기능을 활용하여 중복된 레이아웃 코드를 관리한다.
- **반응형 디자인**: 다양한 디바이스 환경에서 정상적으로 동작하도록 레이아웃을 구성한다.
- **한글 주석**: 각 템플릿의 섹션별 역할을 한글 주석으로 설명한다.

## 테스트
- 실제 페이지 렌더링 시 레이아웃이 정상적으로 적용되는지 확인한다.

## 의존성
- 내부: 없음
- 외부: Mustache Template Engine, Bootstrap 등의 CSS 프레임워크 (header.mustache 참조)
