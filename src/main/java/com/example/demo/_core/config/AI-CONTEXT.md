# config

## 목적
애플리케이션 전반에 적용되는 각종 환경 설정 및 Bean 정의를 포함하는 패키지입니다.

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `AppConfig.java` | 공통적인 Bean 등록 및 설정 (예: 패스워드 인코더 등)을 담당하는 설정 클래스 |

## 하위 디렉토리
- 없음

## AI 작업 지침
- **Bean 관리**: 필요한 설정은 `@Configuration` 클래스 내에 Bean으로 등록하여 관리한다.
- **주석**: 설정의 목적과 구체적인 내용에 대해 한글로 상세히 기술한다.
- **최소 설정**: 불필요한 설정은 제거하고 꼭 필요한 구성 요소만 포함하도록 유지한다.

## 테스트
- 등록된 Bean이 컨텍스트에 정상적으로 로드되는지 단위 테스트로 확인한다.

- 내부: 없음
